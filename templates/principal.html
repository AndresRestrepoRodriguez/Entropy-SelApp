Bienvenido a Entropy-SelApp


Esta aplicación proporciona un entorno Web intuitivo para llevar a cabo la primera iteración de selección de atributos tomando la entropía como medida de ranking para los atributos de un conjunto de datos. A partir de los tipos de datos de los atributos, se aplicarán los métodos de distancia Euclideana, distancia Hamming y cuando sea necesario se discretizará el conjunto de datos a partir de la técnica de ChiMerge. A partir de ello, la plataforma realizará la sugerencia de cual atributo se debería remover del conjunto de datos con base en su valor de entropía.
